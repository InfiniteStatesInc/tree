# SVG 导出指南

## 核心原则

1. **枝干（所有木质部分）合为一个 SVG** — 这是形变动画的主体
2. **花/花冠每团单独导出** — 用于独立的缩放和飘动动画
3. **每帧都要一个「整棵」** — 所有部件保持原位合并，用于自动坐标提取
4. **所有帧的树根底部对齐同一水平线**

---

## 各帧拆分清单

### F1-种子（最小的芽）

```
F1-种子/
  F1-整棵.svg      ← 全部合在一起导出
  F1-枝干.svg      ← 细茎（就那根小棍）
  F1-叶左.svg      ← 左边的小绿叶
  F1-叶右.svg      ← 右边的小绿叶
```

切割说明：
- 枝干：只有那根细茎，从根部到顶
- 叶子：每片叶子单独，沿茎切开

---

### F2-小芽（带花苞）

```
F2-小芽/
  F2-整棵.svg
  F2-枝干.svg      ← 茎（比 F1 稍粗稍高）
  F2-叶左.svg
  F2-叶右.svg
  F2-花苞.svg      ← 顶部那个小粉色花苞
```

切割说明：
- 枝干：茎部分，不含花苞
- 花苞：顶部的粉色小团，单独切出

---

### F3-分叉（开始三分枝）

从这帧开始，枝干分为三组，但**导出时合为一个 SVG**。

```
F3-分叉/
  F3-整棵.svg
  F3-枝干.svg      ← 主干 + 左枝 + 右枝，合为一个 SVG
  F3-花A.svg       ← 中间/顶部的花团
  F3-花B.svg       ← 左枝上的花团
  F3-花C.svg       ← 右枝上的花团
```

**枝干切割要点：**
- 把主干、左枝组、右枝组的所有木质部分（棕色的）选中，一起导出为一个 `F3-枝干.svg`
- 不含任何粉色花、不含绿叶
- 这是 flubber 形变的核心对象，必须是一个连通的路径

**花团切割要点：**
- 每一团粉色的花/花苞单独导出
- 花团按位置编号：A=中/上, B=左, C=右
- 后续帧保持同样的位置编号（同一位置的花从 F3 到 F6 会越长越大）

---

### F4-小树

```
F4-小树/
  F4-整棵.svg
  F4-枝干.svg      ← 全部木质部分合为一个
  F4-花A.svg       ← 中/上位置的花团
  F4-花B.svg       ← 左侧位置的花团
  F4-花C.svg       ← 右侧位置的花团
  F4-花D.svg       ← 如果有第4团的话
```

枝干切割同 F3：所有棕色木质合一。花团按位置对应 F3 的 A/B/C 编号。

---

### F5-中树

```
F5-中树/
  F5-整棵.svg
  F5-枝干.svg      ← 全部木质部分合为一个
  F5-花A.svg       ← 上方主花冠
  F5-花B.svg       ← 左侧花冠
  F5-花C.svg       ← 右侧花冠
  F5-花D.svg       ← 额外花冠（如果有）
  F5-花E.svg       ← 额外花冠（如果有）
```

此时花团开始变大，可能重叠。**即使重叠也要分开导出**——按「哪团是哪团」你自己的感觉来切。

---

### F6-满开

```
F6-满开/
  F6-整棵.svg
  F6-枝干.svg      ← 全部木质部分合为一个（包含所有可见的树枝）
  F6-花A.svg       ← 最大的主花冠
  F6-花B.svg       ← 左侧花冠
  F6-花C.svg       ← 右侧花冠
  F6-花D.svg
  F6-花E.svg
  F6-花F.svg       ← 按实际花冠数量来
```

**满开状态花冠很多很大**，可能有 5-7 团。每团单独导出，哪怕它们视觉上重叠。

---

## 导出设置

- 格式：SVG
- 导出时不要 outline stroke（保持 fill）
- 不要 flatten（保留路径信息）
- Figma 导出选项里取消勾选 "Include 'id' attribute"

## 花团位置编号对照

为了让同一位置的花跨帧对应，统一编号：

```
        花A（中/上）
       /    \
    花B      花C
   (左)     (右)
      \    /
    花D  花E ...（额外位置，F5/F6 才有）
```

**关键**：F3 的花A 和 F6 的花A 应该大致在树上同一个位置（都在中上方），这样代码端可以做这团花从小到大的连续生长。

---

## ⚠️ 最重要的一条

**枝干必须合为一个 SVG**。不要分开导出主干、左枝、右枝——代码端需要把它当作一个整体来做形变。分开了形变会断裂。
